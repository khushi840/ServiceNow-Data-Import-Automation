# ServiceNow Data Import Automation

## Project Overview
This project demonstrates the automation of data import and relationship mapping using the ServiceNow platform. It explains how external datasets such as CSV or Excel files can be integrated into ServiceNow using Import Sets and Transform Maps.

## Tools and Technologies
- ServiceNow
- Import Sets
- Transform Maps
- Dot Walking

## Description
The project focuses on automating the process of importing employee data into ServiceNow and establishing relationships between different records such as employees, departments, and managers. This automation improves data accuracy, reduces manual effort, and enhances enterprise data management.
